# Safe Garden Home Daycare

Landing page for Safe Garden Home Daycare (Coquitlam, BC).

## Overview
This is a simple, responsive landing page designed to increase tour bookings and inquiries.

## Features
- Mobile-friendly design
- Clear call-to-action (Check Availability / Book a Tour)
- Parent testimonials
- About the owner (Chonon)
- Google Forms integration for inquiries
